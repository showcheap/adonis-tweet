# AdonisJs Tweet
Simple application built with AdonisJS + Bootstrap 4

# Install
* `git clone https://github.com/showcheap/adonis-tweet.git`
* `cd adonis-tweet`
* `npm install`
* `./ace migration:run`
* `./ace db:seed`
* `npm run serve:dev`
* Open your browser on `http://localhost:3333`
